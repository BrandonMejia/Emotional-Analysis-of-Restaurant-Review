README

-------------------------------------------------------------------------------------------------
Esta aplicacion utiliza una API de Watson de Analisis de Sentimiento para
realizar el analisis de sentimiento de los comentarios los clientes de un restaurante.

Pasos para ejecutar el sistema:
    1) Tener instalado python 3.4 o superior (No ha sido probado con versiones inferiores)
    2) Instalar la libreria ibm-watson. Utiliza la terminal de comandos o CMD y siga estos pasos:
        2.1) cd 'Ruta exacta del proyecto Sentiment analysis of restaurant reviews'
        2.2) pip install --upgrade "ibm-watson>=6.0.0"
	3) Para finalizar, ejecute
	    python main.py
	  en el mismo lugar donde ejecuto el anterior comando.
